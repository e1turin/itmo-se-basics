## Идея составления ТЗ
Считаем, что мы хотим реализовать сайт для информационного амурской обласи с целью централизованно информировать людей о всех событиях просходящих в регионе. При этом, чтобы увеличить посещаемость сайта, мы предложим тематические подборки событий для каждого типа посетителя сайта, по сути это главная составляющая нашего сайта -- подборки новостей.
